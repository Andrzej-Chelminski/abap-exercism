# ABAP Exercism Solutions 🚀

This repository contains my solutions to the exercises from the [Exercism](https://exercism.org/tracks/abap) platform.

The main goal of this project is to gain hands-on experience with modern ABAP, improve my algorithmic thinking, and practice writing clean, readable code. By solving these tasks, I am also preparing for the SAP Certified - Backend Developer - ABAP Cloud certification.

## 📂 Project Structure

The codebase is maintained in an SAP system (Trial environment) and synchronized using the **abapGit** plugin.

The package structure is organized as follows:
* `Z_EXERCISM` - The main package aggregating all tasks.
  * `Z_EXERCISM_TASK_1` - Subpackage containing the solution for the first task.
  * `Z_EXERCISM_TASK_2` - Subpackage containing the solution for the second task, etc.

## 🛠️ Technologies & Tools
* **Language:** ABAP (focusing on modern ABAP Cloud syntax)
* **Environment:** SAP BTP ABAP Environment (Trial)
* **IDE:** Eclipse with ABAP Development Tools (ADT)
* **Version Control:** abapGit
